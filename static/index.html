添加供应商和模型

                        
先导入一个预设，再补供应商地址、密钥和模型。一个供应商可以放多个不同能力的模型。

                    

                    

                        
                        
                    

                

                

                

                    
缺少端口、访问标识、超时等信息时，会优先使用安全默认值。

                    
每个模型都可以单独设置上下文、接口、图片能力和推理深度策略。

                    
以后修改时直接回到供应商页，不需要重跑整个向导。

                

            

            

                

                    

                        
设置路由和图片兜底

                        
后台调整模型顺序后，会在当前会话结束后切到新的选择；图片可指定全局兜底供应商。

                    

                    

                        
                        
                    

                

                

                

                    
智能路由按当前顺序选择，健康状态异常时会优先避开。

                    
纯文本代理不会被标成“全部不支持”，而是提示配置媒体供应商。

                    
纯原生代理要确认图片接口是否也经过同一个供应商地址。

                

            

            

                

                    

                        
费用与币种

                        
可手动设置汇率；在线汇率暂时关闭，避免不确定结果。

                    

                    
                

                

                    
用量、额度和余额提醒

                    
Token 用量以 Codex 输出记录为准；额度和余额只有在能确认来源时才开启。

                    

                        
有 Codex 事件时优先使用事件；没有时只显示可确认的本地记录。

                        
额度需要官方接口、内置脚本或你导入的脚本。

                        
余额优先用官方扣费；没有价格时不估算费用。
